# Flow Ecosystem 开发计划

> 最后更新：2026-05-08 | 版本：V7.7

---

## 一、项目概述

### 1.1 核心理念
- 🪞 **认知镜像** —— 只帮用户看见自己的节奏，不做监控机器
- 🎛️ **AI 调音** —— 根据用户状态自动调整语气、深度、推进节奏
- 🔧 **破局指南** —— 工具不是推荐出来的，是从今天的损失里长出来的

### 1.2 架构设计（Hybrid 模式）

```
┌─────────────────────────────────────────────────────────────┐
│  Skill 接口层（OpenClaw 入口）                              │
│  ├── /flow_report 命令 → 触发认知体检报告                   │
│  └── 自然语言意图识别 → 识别用户召唤                     │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  Plugin 核心层（业务逻辑）                                  │
│  ├── session_analyzer.py    → LLM 4维判定                  │
│  ├── state_distiller.py     → 12条规则匹配 + 调音         │
│  ├── report_generator.py    → /flow_report 组装 + LLM润色    │
│  ├── daily_aggregator.py    → 日级趋势计算（待实现）        │
│  ├── memory_manager.py      → MEMORY.md锚点管理（待实现）      │
│  └── brave_search.py        → 联网动态工具搜索             │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  Core 基础设施层                                              │
│  ├── db.py              → SQLite 连接                       │
│  ├── llm_client.py       → LLM客户端封装                  │
│  ├── config.py          → 配置管理                         │
│  └── vector_layer.py     → sentence-transformers 向量层     │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  Data 数据层                                              │
│  └── flow_ecosystem.db  → SQLite 数据库                     │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  MEMORY.md（OpenClaw 共享记忆）                              │
│  ├── FLOW_STYLE 锚点    → AI实时调音参数                    │
│  ├── FLOW_STATUS 锚点   → 当前状态快照（待实现）              │
│  ├── GOALS 锚点        → 活跃目标列表（待实现）              │
│  ├── PATTERNS 锚点      → 行为模式记录（待实现）              │
│  └── COGNITION 锚点     → 认知进化记录（待实现）              │
└─────────────────────────────────────────────────────────────┘
```
